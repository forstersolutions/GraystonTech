MyPokerMaps / StackTrax

Live poker discovery, club onboarding, and operator workflow in one ecosystem.

QDrop / QrystalDrop

Quantum-safe secure file transfer with simple encrypt and decrypt flows.

QDrop is live at qdrop.vercel.app. The scanned MVP includes an encrypt/upload panel, a download/decrypt panel, session sampling details, AES-GCM encryption positioning, expiry time, max downloads, and auto-deletion behavior.

  • Encrypt and upload flow with drag-and-drop file selection
  • Secure link handoff for download and decrypt workflow
  • Expiry and max-download controls for temporary transfer access
  • Product path toward governed file movement and enterprise-grade evidence
